# NasaProject

## Description
This project is build after i learned Nodejs course on Udemy. We use https://github.com/arwes/arwes to setup UI of app, use SpaceX Rest API to get space mission data. My work is to build back-end for this app.

## Use

You need to run ```npm intall``` first to install all dependency for this app

Next, Run```npm run deploy``` to use it.
